# 医美价目表批量维护工具

专为单店或小型医美机构运营负责人设计的桌面端工具，强调**极简、离线、批处理**。

## ✨ 核心特性

- 🗂️ **5 大功能窗口**：导入表格 / 价格校验 / 批量调整 / 打印预览 / 备份恢复
- 📥 **智能导入**：拖拽 Excel 文件，自动识别列字段和项目分类
- ⚠️ **冲突处理**：同名项目不同价格自动标记颜色，逐条或批量解决
- ✅ **价格校验**：会员价高于原价检测、低于成本价拦截
- ⚡ **批量调整**：
  - 按百分比统一上调（如：光电类 +10%）
  - 按固定金额下调（如：清洁补水体验价 -¥30）
  - 指定品牌耗材批量加价（如：乔雅登 +¥100）
  - 批量设置前台展示名 / 内部核算名
- 🖨️ **打印导出**：A4 打印版价目表 / 咨询师精简版 / Excel / CSV
- 🔙 **版本管理**：自动保存修改快照，误操作一键回滚
- 🔒 **完全离线**：所有数据本地存储，无需网络

## 🚀 快速开始

### 方式一：一键启动（推荐）

双击 `启动.bat`，脚本会自动：
1. 创建虚拟环境 `venv`
2. 安装所有依赖
3. 启动应用

### 方式二：手动启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行
python main.py
```

## 🎯 使用流程

### 第一步：导入表格
1. 打开「导入表格」页面
2. 将 Excel / CSV 文件拖入拖拽区（或点击选择）
3. 系统自动识别：项目名称、分类、原价、会员价、医生费、耗材费、备注
4. 若有同名项目价格冲突，点击「批量解决」或「逐条处理」
5. 点击「确认导入」

### 第二步：价格校验
1. 自动检测：会员价 > 原价 / 低于成本价 / 重名项目
2. 红色：严重问题；黄色：需要注意；绿色：正常
3. 支持「一键修复」功能
4. 双击单元格可直接编辑修改

### 第三步：批量调整
- **百分比调整**：选择分类 → 输入涨幅 → 预览 → 应用
- **固定金额**：选择分类 → 调整金额 → 仅体验价可选
- **品牌加价**：输入品牌关键词 → 固定或百分比加价
- **名称设置**：批量添加前缀/后缀、查找替换

### 第四步：打印与导出
1. 编辑标题 / 门店信息
2. 选择导出类型（正式版 / 无会员价 / 咨询师版）
3. 筛选需要的分类
4. 预览 PDF 或导出 Excel / CSV

### 第五步：备份管理
- 每次保存自动创建快照
- 支持手动创建、重命名、导出快照
- 选择历史版本 → 点击「回滚到此版本」
- 回滚前自动备份当前版本

## 🔧 常见问题

**Q: 启动时提示找不到 Python？**
A: 请先安装 Python 3.9+，安装时勾选「Add Python to PATH」。

**Q: 导入时列识别不正确？**
A: 确保 Excel 第一行是列标题，名称尽量接近规范字段。可手动编辑修改。

**Q: PDF 导出中文显示为方框？**
A: 请确认系统已安装「微软雅黑」或「宋体」字体（Windows 默认自带）。

**Q: 如何迁移数据到另一台电脑？**
A: 复制整个项目文件夹（含 `app/data` 和 `app/backups` 目录）即可。
